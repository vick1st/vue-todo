# 🚀 Vue.js Todo-List

Projeto para aprender Vue.js 3 na prática construindo uma Todo-List completa.

## 🎯 Objetivo

Dominar os fundamentos do Vue.js 3:

- Componentes, Props & Reatividade
- Vue Router & Pinia
- Boas práticas

## 🚀 Como rodar

```sh
npm install
npm run dev
```

## 🛠️ Build

```sh
npm run build
```

Feito para estudo ❤️
